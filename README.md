# Legal-Contract-Intelligence-using-NLI

• Developed a document-level Natural Language Inference (NLI) system for legal contracts to classify entailment,
contradiction, and neutral relationships with evidence span extraction.  
• Fine-tuned transformer models including BERT (Span-based NLI) and Phi-3 LLM using prompt-based completion
for improved contextual reasoning over complex legal text.  
• Designed and evaluated multiple approaches including classical baselines (TF-IDF, cosine similarity) and deep
learning models, performing detailed EDA and experimentation across dataset splits.  
• Handled class imbalance and optimization challenges using custom loss strategies and model tuning, improving
robustness on real-world legal datasets
